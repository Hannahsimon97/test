mixtape

a website where users can make playlists and send them to friends as a mixtape

Big Picture

mixtape triggers nostaligia for the early 2000's by digitizing the classic "mixtape" or CD-ROM.

  • Users connect with their spotify account to transfer playlists to the mixtape website.

  • From there, users can decorate a digital CD containging their playlist using their own writing, stickers, stamps just like the good ol' days.

  • Users can send the CD mixtapes to friends via all sharing platforms. The catch is, the songs aren't labeled, just like an actual CD, so you don't know what jams are coming next

Coming Soon

mixtape is a passion project started by Hannah Simon and Cooper Bodary.

mixtape has been in the works for the last few months. We hope to launch by the end of summer.

All of the code for mixtape will be available in my github when the project is finished.

mixtape was made using HTML, CSS, Java, JavaScript, Python and SQL
